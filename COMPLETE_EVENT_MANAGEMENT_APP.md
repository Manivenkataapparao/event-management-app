# Bellcorp Event Management Application

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js
- **Database:** MongoDB (preferred) or SQLite

## Features

### Authentication
- User registration
- User login
- Protected routes for authenticated users

### Event Listings
Browse and manage events with the following details:
- Event name
- Organizer
- Location
- Date & time
- Description
- Available seats/capacity
- Category or tags

**User Actions:**
- Browse available events
- View detailed event information
- Register for an event
- Cancel event registration

### Event Discovery Experience
- Explore large event collections efficiently
- Retrieve event data dynamically while browsing
- Search events using flexible text queries
- Filter events by date, location, or category
- Maintain browsing state across navigation
- Handle changing event availability

### User Dashboard
- List of registered events
- Upcoming events summary
- Past event history
